# conscientizacao-ruas

Projeto front-end simples para divulgação e conscientização sobre a população em situação de rua. Conteúdo baseado no trabalho acadêmico "O DESENVOLVIMENTO DO SER HUMANO NAS RUAS".

- Arquivos:
  - `index.html` — página única com seções: Hero, Infância e Ruptura, Saúde Mental e Sobrevivência, Estigma.
  - `style.css` — estilos responsivos em tons sóbrios de azul, cinza e branco; uso de Grid e Flexbox.
  - `script.js` — rolagem suave, fade-in por IntersectionObserver e indicação de link ativo.

## Como usar

1. Abrir `index.html` no navegador para visualizar a página.
2. Para publicar no GitHub Pages, ative o branch `main` e habilite Pages nas configurações do repositório.
